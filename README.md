# assign-6-Tkinter-Calculator
Simple Calculator Using Python Tkinter

This project is a basic calculator application developed using Python and the Tkinter library. 
It performs simple arithmetic operations such as addition, subtraction, multiplication, and division.
The calculator is designed in a beginner-friendly way using simple functions and button placement with the place() method.

Features:

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Equal (=) button to display the result
* Clear button to reset the input field
* Handles division by zero by displaying "Error"

How to Run the Project:

1. Make sure Python is installed on your system.
2. Open the project folder.
3. Open the file 
4. Run the file
5. The calculator window will open and you can start using it.

How It Works:
The Entry widget is used to accept user input. Number buttons call the click() function to display numbers in the entry box. Operator buttons store the first number and the selected operation. When the Equal button is pressed, the program performs the calculation and displays the result. The Clear button removes all input from the entry field.
